# Website Creation (AltSwitcher Landing Page)

## Architecture & Frameworks
- **Tech Stack:** Vanilla HTML/CSS/JS or a lightweight Vite framework.
- **Styling:** Vanilla CSS focusing on modern UI design (glassmorphism, vibrant colors, dark mode, smooth animations).
- **Target Audience:** macOS users.

## Core Requirements
1. **Modern Premium Design:**
   - Use high-quality typography (Inter, Roboto, or MacOS native fonts).
   - Implement micro-animations for interactivity.
   - Avoid standard colors; use tailored HSL palettes.

2. **OS Detection:**
   - Write JavaScript to detect the user's OS.
   - Tailor the call-to-action button (e.g., "Download for macOS" vs "Available only on macOS").

3. **SEO Best Practices:**
   - Include descriptive `<title>` and `<meta name="description">` tags.
   - Use a single `<h1>` for the page title.
   - Add Open Graph (OG) tags for social sharing.

4. **Content Structure:**
   - Hero section with the primary value proposition (e.g., "Never retype your text again").
   - Features section (Auto-switch, Manual switch, App Blacklist).
   - Changelog/Release History.
   - Download section.
